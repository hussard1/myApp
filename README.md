# myApp

개인 공부용 프로젝트

- 백엔드 : kotlin, gradle, spring boot2, spring JPA
- DB: 몽고DB (클러스터 node-3, replica-1)
- 프론트 엔드 : html/css sass, react.js, context api 
- git hub 
- 배포 : CI 젠킨스, CD 쿠버네티스 master-1 worker-3 (우선은 수동으로 EKS 검토 고려)
- 로깅 : elastic search, fluentd, kibana
- apm : pinpoint
- prometheus, grafana
- 캐쉬 : elastic cache 사용
- 데이터 파이프 라인 : kafaka
